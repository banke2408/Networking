Physical Network Path
Data Encapsulation
Headers Added Layer by Layer
As data travels DOWN the sender's OSI stack, each layer wraps the data with its own header (and sometimes a trailer). This is called encapsulation. At the receiver, each layer peels off its header — decapsulation.
🧅 Think of it like wrapping a letter in envelopes. Layer 7 puts the letter in an envelope. Layer 4 puts that in another envelope with a port number. Layer 3 adds another with IP addresses. Layer 2 adds the MAC address envelope. Each envelope is read and removed by the matching layer on the receiver side.
